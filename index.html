MultiLogicTrade

FINRESP-калькулятор, реальная и фейковая торговля T-Bank, справки MultiLogic.

Онлайн (ПК, планшет, телефон) калькулятор на GitHub Pages · справка — не нужен zip и локальный сервер.

Локально на ПК: не открывайте HTML двойным кликом (file://) — MOEX не загрузится. Запустите serve-calculator.ps1 в корне репозитория (откроется http://127.0.0.1:8765/…). Через file:// работают песочница и T-Bank, но не загрузка свечей с MOEX.

Android / планшет (скачанный zip) Не открывайте index.html из «Загрузок» — адрес content://… даёт браузеру только один файл, ссылки на FinrespCalculator/… не работают. Распакуйте папку и откройте сайт по http://: Termux — cd ~/storage/downloads/MultiLogicTrade, затем python -m http.server 8765, в Chrome — http://127.0.0.1:8765/index.html. Или используйте эту страницу на GitHub Pages.

  • Справка по роботу — строки логики L1…L5
  • Рынок в терминах многочленов
